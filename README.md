# ユーザー登録機能サンプル

## 概要
本リポジトリは、FlaskとMySQLを用いたユーザー登録機能のサンプルです。  
実務で開発したシステムの一部を切り出し、ポートフォリオ用に簡略化しています。  
個人情報や機密情報については、すべてダミーデータに置き換えています。

開発環境はDockerを用いて構築しました。  
実際の運用では、小規模な飲食店のローカル環境に導入し、顧客のPCへリモート接続して環境構築および初期設定を行いました。


## 主な機能
1. **ユーザー登録**  
   - 名前、パスワード、時給を入力して新規ユーザーを登録
   - パスワードはハッシュ化して安全に保存

2. **ユーザー一覧表示**  
   - 登録済みのユーザー情報（ID、名前、時給）を取得・表示

3. **ユーザー削除**  
   - 複数ユーザーをまとめて削除可能
   - フロントエンドからチェックボックスで削除対象を選択可能
   - 
4. **打刻機能**  
   -従業員の出社及び、退社を記録
   
5. **ユーザーの打刻情報の変更昨日**  
   　誤って打刻してしまったユーザーのために、打刻情報を変更する機能(管理者のみが可能)
